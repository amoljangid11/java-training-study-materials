HTML Templates : HTML <template> is a special HTML element used to store reusable HTML content that is not rendered immediately on the webpage.

It is useful when you want to:

Dynamically create UI components
Reuse HTML blocks
Improve performance
Build cards, tables, modals, lists, etc.
Work with JavaScript frameworks and Web Components

1. What is an HTML Template?

The <template> tag contains HTML code that is hidden until JavaScript uses it.

Syntax
<template id="templateName">
    <!-- Hidden HTML -->
</template>

The content inside:

Does NOT display on the page initially
Does NOT execute scripts immediately
Can be cloned and inserted using JavaScript

2. Why Use Templates?

Without templates:

<div class="card">
   <h2>John</h2>
   <p>Developer</p> CSS
</div>

<div class="card">
   <h2>Mike</h2>
   <p>Tester</p>
</div>

This creates repeated code.

With templates:

Write once
Reuse many times
Dynamically inject data

3. Basic HTML Template Example

Example

<!DOCTYPE html>
<html>
<head>
    <title>HTML Template Example</title>
</head>
<body>

<h1>HTML Template Demo</h1>

<template id="myTemplate">
    <div class="card">
        <h2>Name</h2>
        <p>Role</p>
    </div>
</template>

<script>

    // Access template
    const template = document.getElementById("myTemplate");

    // Clone template content
    const clone = template.content.cloneNode(true);

    // Add clone to body
    document.body.appendChild(clone);

</script>

</body>
</html>

5. Dynamic Template Example

Create employee cards dynamically. //DOM

Complete Example
<!DOCTYPE html>
<html>
<head>
    <title>Employee Cards</title>

    <style>

        .card{
            border:1px solid gray;
            padding:15px;
            margin:10px;
            width:250px;
            border-radius:10px;
            background:#f5f5f5;
        }

    </style>

</head>
<body>

<h1>Employee List</h1>

<div id="container"></div>

<!-- Template -->
<template id="employeeTemplate">

    <div class="card">

        <h2 class="name"></h2>

        <p class="designation"></p>

        <button>View Profile</button>

    </div>

</template>

<script>

    const employees = [ // Array of Object

        {
            name:"Amol",
            designation:"Java Developer"
        },

        {
            name:"Ravi",
            designation:"Angular Developer"
        },

        {
            name:"Sneha",
            designation:"Python Developer"
        }

    ];

    const template = document.getElementById("employeeTemplate");

    const container = document.getElementById("container");

    employees.forEach(emp => {

        // Clone template
        const clone = template.content.cloneNode(true);

        // Modify cloned content
        clone.querySelector(".name").textContent = emp.name;

        clone.querySelector(".designation").textContent =
            emp.designation;

        // Append to container
        container.appendChild(clone);

    });

</script>

</body>
</html>

4. Geolocation API

What is Geolocation?

Geolocation API allows browsers to access user location.

Uses :

Maps
Delivery apps
Tracking
Nearby services

Important Notes :

Requires user permission

Works better with HTTPS

Geolocation Example :

<!DOCTYPE html>
<html>
<head>
    <title>Geolocation Example</title>
</head>

<body>

<h1>Get Current Location</h1>

<button onclick="getLocation()">
    Get Location
</button>

<p id="result"></p>

<script>

function getLocation() {

    if (navigator.geolocation) {

        navigator.geolocation.getCurrentPosition(
            showPosition,
            showError
        );

    } else {
        document.getElementById("result").innerHTML =
            "Geolocation not supported";
    }
}

function showPosition(position) {

    document.getElementById("result").innerHTML =
        "Latitude: " + position.coords.latitude +
        "<br>Longitude: " + position.coords.longitude;
}

function showError(error) {

    switch(error.code) {

        case error.PERMISSION_DENIED:
            alert("Permission denied");
            break;

        case error.POSITION_UNAVAILABLE:
            alert("Location unavailable");
            break;

        case error.TIMEOUT:
            alert("Request timeout");
            break;

        default:
            alert("Unknown error");
    }
}

</script>

</body>
</html>

1. Inline CSS

Inline CSS is written directly inside an HTML element using the style attribute.

Syntax

<tag style="property:value;">

Example of Inline CSS

<!DOCTYPE html>
<html>
<head>
    <title>Inline CSS Example</title>
</head>

    <h1 style="color:blue;">Welcome to CSS</h1>

    <p style="font-size:20px; color:red;">
        This paragraph uses inline CSS.
    </p>

    <button style="background-color:green; color:white; padding:10px;">
        Click Me
    </button>

</body>
</html>

Output Explanation

color:blue; → Changes text color
font-size:20px; → Changes text size
background-color:green; → Button background color
padding:10px; → Space inside button

Advantages of Inline CSS

Easy for small styling
Quick testing
Highest priority among CSS types

Disadvantages of Inline CSS

Difficult to maintain
Repeated code
Not reusable
Makes HTML messy

CSS Box Model

Every HTML element is treated as a rectangular box.

The CSS Box Model contains:

Content
Padding
Border
Margin

Structure of Box Model

 -------------------------
|         Margin          |
|  ---------------------  |
| |       Border        | |
| |  -----------------  | |
| | |     Padding     | | |
| | |  -----------    | | |
| | | |  Content |    | | |
| | |  -----------    | | |
| |  -----------------  | |
|  ---------------------  |
 -------------------------

Box Model Components

1. Content

Actual text or image inside element.

Example:

<p>Hello World</p>

2. Padding

Space between content and border.

padding: 20px;

3. Border

Line around padding/content.

border: 5px solid black;

4. Margin

Space outside the border.

margin: 30px;

Complete Box Model Example

<!DOCTYPE html>
<html>
<head>
    <title>Box Model Example</title>

    <style>

        .box {

            width: 300px;
            height: 150px;

            background-color: lightblue;

            padding: 20px;

            border: 5px solid blue;

            margin: 40px;

            font-size: 20px;
        }

    </style>
</head>

<body>

    <div class="box">
        This is CSS Box Model Example
    </div>

</body>
</html>

Understanding the Values

width: 300px;

Content width = 300px

padding: 20px;

20px space inside border

border: 5px solid blue;

5px border around element

margin: 40px;

40px space outside border

Total Width Calculation

Formula:

Total Width =

Content Width
+ Left Padding + Right Padding
+ Left Border + Right Border
+ Left Margin + Right Margin

Using above example:

300
+ 20 + 20
+ 5 + 5
+ 40 + 40

= 430px

Box Sizing

Normally, padding and border increase total size.

To avoid this, use:

box-sizing: border-box;

Example

<!DOCTYPE html>
<html>
<head>
<style>

.box {
    width: 300px;
    padding: 20px;
    border: 5px solid red;

    box-sizing: border-box;

    background-color: yellow;
}

</style>
</head>

<body>

<div class="box">
    Width remains 300px
</div>

</body>
</html>

Background Color Applied	No			Yes
Affects Outside Spacing	Yes			No

Preprocessors solve these issues.

CSS Grid Layout :

CSS Grid Layout is a 2-dimensional layout system used to design web pages using rows and columns.

It helps developers create:

Responsive layouts
Dashboards
Gallery layouts
Website structures
Complex UI designs

1. Why CSS Grid?

Before Grid:

Developers used floats
Flexbox was used for 1D layouts only column

Grid solves:

Row + column layouts together

Alignment issues

Complex positioning

2. Basic Terminologies

Term		Description

Grid Container	Parent element
Grid Item		Child elements
Grid Row		Horizontal lines
Grid Column	Vertical lines
Grid Gap		Space between items   1  2  3  4

3. Basic Grid Structure

HTML

<div class="container">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
    <div class="item">4</div>
</div>

CSS

.container {
    display: grid;
}

display: grid converts the container into a grid container.

4. Creating Columns

CSS

.container {
    display: grid;

    grid-template-columns: 200px 200px 200px;
}

This creates:

3 columns

Each width = 200px

5. Creating Rows

.container {
    display: grid;

    grid-template-columns: 200px 200px 200px;

    grid-template-rows: 100px 100px;
}

Creates:

2 rows

Each height = 100px

Animation and Transitions in CSS

CSS Animations and Transitions are used to create smooth visual effects in web applications.

Transition → Changes property values smoothly from one state to another.
Animation → Creates complex movements and multi-step effects.
1. CSS Transitions

A transition occurs when a CSS property changes value.

Example:

Button color changes smoothly when hovered.
Image zooms slowly.
Syntax of Transition
transition: property duration timing-function delay;
Property	Description
property	CSS property to animate
duration	Time for animation
timing-function	Speed curve
delay	Delay before start
Example 1 — Simple Button Transition
HTML
<!DOCTYPE html>

    background-color: red;

How it Works

When mouse hovers:

background changes from blue → red
change happens in 0.5 seconds

Function	Description

ease	Slow-fast-slow

ease-out	Slow end

Responsive design solves these problems.

@media screen and (max-width: 768px){

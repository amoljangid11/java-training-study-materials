Introduction to RDBMS: 1. What is a Database?

A database is an organized collection of data stored electronically so that it can be easily accessed, managed, and updated.

Real-Life Examples

Banking systems

Hospital systems

Railway reservation systems

Example

2. What is RDBMS?

RDBMS = Relational Database Management System 

An RDBMS is software used to create, manage, and manipulate relational databases.

It stores data in the form of tables.

3. Why is it called “Relational”?

Because tables are related to each other using relationships.

Example:

One student can enroll in many courses
One department can have many employees

These relationships are managed using keys. Foreign Key

5. Features of RDBMS

a) Table-Based Storage

Data is stored in rows and columns.

b) Relationships

c) Data Integrity

d) Security

6. Basic Terminologies in RDBMS

7. Types of Relationships in RDBMS

a) One-to-One (1:1)

One person → One passport

b) One-to-Many (1:M)

One department → Many employees

c) Many-to-Many (M:M)

Many students → Many courses

This requires a junction table.

11. What is a Transaction?

A transaction is a logical unit of work.

Example:

Bank money transfer:

Debit from Account A
Credit to Account B

Both operations must succeed together.

12. ACID Properties

ACID ensures reliable transactions.

A → Atomicity

Either complete fully or not at all.

C → Consistency

Database remains valid.

I → Isolation

Transactions do not interfere.

D → Durability

Committed data remains permanent.

Select Statement : The SELECT statement is used to retrieve data from one or more tables in a MySQL database.

It is one of the most important SQL commands.

Basic Syntax

SELECT column1, column2, ...
FROM table_name;

OR

SELECT * FROM table_name;

SELECT → specifies columns to retrieve
FROM → specifies table name
* → selects all columns

INNER JOIN Example : 

SELECT e.emp_name,
       e.salary,
       d.dept_name
FROM employees e
INNER JOIN departments d
ON e.department = d.dept_name;

LEFT JOIN Example :

select e.emp_id, e.emp_name, d.dept_name from employees e left join departments d on e.department = d.dept_name;

RIGHT JOIN Example :

select e.emp_id, e.emp_name, d.dept_name from employees e right join departments d on e.department = d.dept_name;

FULL OUTER JOIN : 


select e.emp_id, e.emp_name, d.dept_name from employees e left join departments d on e.department = d.dept_name union select e.emp_id, e.emp_name, d.dept_name from employees e right join departments d on e.department = d.dept_name;

CROSS JOIN :

select e.emp_name, d.dept_name from employees e cross join departments d;

SELF JOIN : 

select e.emp_name, m.salary from employees e left join employees m on e.emp_id = m.emp_id;

Introduction to JDBC : 1. What is JDBC?

JDBC (Java Database Connectivity) is a Java API used to connect Java applications with databases.

Using JDBC, a Java program can: CRUD

Connect to a database
Execute SQL queries
Insert, update, delete records
Retrieve data from tables
Handle transactions

JDBC acts as a bridge between:

Java Application
Database (MySQL, Oracle, PostgreSQL, SQL Server, etc.)

Without JDBC:

Java applications cannot directly communicate with databases.

With JDBC:

import java.sql.*;

public class PreparedDemo {

    public static void main(String[] args) {

        try {

            Class.forName("com.mysql.cj.jdbc.Driver");

            Connection con = DriverManager.getConnection(
                    "jdbc:mysql://localhost:3306/company",
                    "root",
                    "password");

            String query =
                    "INSERT INTO employee VALUES(?,?,?)";

            PreparedStatement ps =
                    con.prepareStatement(query);

            ps.setInt(1, 101);
            ps.setString(2, "David");
            ps.setDouble(3, 75000);

            int rows = ps.executeUpdate();

            System.out.println(rows + " row inserted");

            con.close();

        } catch(Exception e) {
            e.printStackTrace();
        }
    }
}

🔟 DAO Class for CRUD Operations

EmployeeDAO.java

package com.jdbc.dao;

import com.jdbc.connection.DBConnection;
import com.jdbc.model.Employee;

import java.sql.Connection;
import java.sql.PreparedStatement;
import java.sql.ResultSet;

public class EmployeeDAO {

    // INSERT OPERATION
    public void insertEmployee(Employee employee) {

        String query =
                "INSERT INTO employee(name, email, salary) VALUES(?,?,?)";

        try {

            Connection connection = DBConnection.getConnection();

            PreparedStatement ps =
                    connection.prepareStatement(query);

            ps.setString(1, employee.getName());
            ps.setString(2, employee.getEmail());
            ps.setDouble(3, employee.getSalary());

            int rows = ps.executeUpdate();

            if(rows > 0) {
                System.out.println("Employee Inserted Successfully");
            }

            connection.close();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    // READ OPERATION
    public void getAllEmployees() {

        String query = "SELECT * FROM employee";

        try {

            Connection connection = DBConnection.getConnection();

            PreparedStatement ps =
                    connection.prepareStatement(query);

            ResultSet rs = ps.executeQuery();

            while(rs.next()) {

                System.out.println(
                        rs.getInt("id") + " "
                        + rs.getString("name") + " "
                        + rs.getString("email") + " "
                        + rs.getDouble("salary")
                );
            }

            connection.close();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    // UPDATE OPERATION
    public void updateEmployee(int id, double salary) {

        String query =
                "UPDATE employee SET salary=? WHERE id=?";

        try {

            Connection connection = DBConnection.getConnection();

            PreparedStatement ps =
                    connection.prepareStatement(query);

            ps.setDouble(1, salary);
            ps.setInt(2, id);

            int rows = ps.executeUpdate();

            if(rows > 0) {
                System.out.println("Employee Updated Successfully");
            }

            connection.close();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    // DELETE OPERATION
    public void deleteEmployee(int id) {

        String query =
                "DELETE FROM employee WHERE id=?";

        try {

            Connection connection = DBConnection.getConnection();

            PreparedStatement ps =
                    connection.prepareStatement(query);

            ps.setInt(1, id);

            int rows = ps.executeUpdate();

            if(rows > 0) {
                System.out.println("Employee Deleted Successfully");
            }

            connection.close();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

            ps.setString(1, emp.getName());
            ps.setString(2, emp.getEmail());
            ps.setDouble(3, emp.getSalary());

            System.out.println(rows + " Record Inserted");

        } catch (Exception e) {
            e.printStackTrace();
        }
    }

        try {

            String sql =
                    "UPDATE employee SET name=?, email=?, salary=? WHERE id=?";

            PreparedStatement ps = con.prepareStatement(sql);

            ps.setString(1, emp.getName());
            ps.setString(2, emp.getEmail());
            ps.setDouble(3, emp.getSalary());
            ps.setInt(4, emp.getId());

package com.jdbc.transaction;

import java.sql.CallableStatement;
import java.sql.Connection;

public class TransactionExample {

    public static void main(String[] args) {

        Connection con = null;
        CallableStatement cs = null;

        try {

            // Get Connection
            con = DBConnection.getConnection();

            // Start Transaction
            con.setAutoCommit(false);

            // Prepare Callable Statement
            cs = con.prepareCall("{call transfer_money(?, ?, ?)}");

            // Set Parameters
            cs.setInt(1, 1);
            cs.setInt(2, 2);
            cs.setDouble(3, 1000);

            // Execute Procedure
            cs.execute();

            System.out.println("Money Transferred Successfully");

            // Commit Transaction
            con.commit();

            System.out.println("Transaction Committed");

        } catch (Exception e) {

            try {

                // Rollback Transaction
                if (con != null) {

                    con.rollback();

                    System.out.println("Transaction Rolled Back");
                }

            } catch (Exception ex) {

                ex.printStackTrace();
            }

            e.printStackTrace();

        } finally {

            try {

                if (cs != null)
                    cs.close();

                if (con != null)
                    con.close();

            } catch (Exception e) {

                e.printStackTrace();
            }
        }
    }
}

package com.jdbc.transaction;

import java.sql.CallableStatement;
import java.sql.Connection;
import java.sql.Types;

public class OutParameterExample {

    public static void main(String[] args) {

        try {

            Connection con = DBConnection.getConnection();

            CallableStatement cs =
                    con.prepareCall("{call get_balance(?, ?)}");

            // IN Parameter
            cs.setInt(1, 1);

            // OUT Parameter
            cs.registerOutParameter(2, Types.DOUBLE);

            cs.execute();

            System.out.println("Balance = " + balance);

            con.close();

        } catch (Exception e) {

    }
